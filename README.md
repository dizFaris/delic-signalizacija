# Delic Signalizacija

Nuxt 4 project scaffolded for further development with PrimeVue, Tailwind CSS and Nuxt UI available in the stack.

## Included

- PrimeVue Nuxt module with Aura theme
- Shared layout and working top navigation
- Valid pages for home, about, services and contact
- Redirect from `/serices` to `/services`
- Lint, typecheck and build scripts

## Development

Install dependencies:

```bash
pnpm install
```

Start the dev server:

```bash
pnpm dev
```

## Validation

```bash
pnpm lint
pnpm typecheck
pnpm build
```
